# Todo List ✅

A feature-rich todo list application with local storage persistence.

## Features
- ✅ Add, edit, and delete tasks
- ✅ Mark tasks as complete
- ✅ Filter tasks (All, Active, Completed)
- ✅ Clear completed tasks
- ✅ Task counter
- ✅ LocalStorage persistence (tasks save automatically)
- ✅ Responsive design
- ✅ Inline editing (click Edit button, type, press Enter)
- ✅ Empty state messages

## How to Use

### 1. Add a Task
- Type your task in the input box (e.g., "Buy groceries")
- Click the **Add** button or press **Enter**

### 2. Complete a Task
- Click the **checkbox** next to the task
- The task will get a strikethrough

### 3. Edit a Task
- Click the **Edit** button
- Type directly in the inline input field
- Press **Enter** to save or **Escape** to cancel

### 4. Delete a Task
- Click the **Delete** button
- Task will be removed immediately

### 5. Filter Tasks
- **All**: Shows all tasks
- **Active**: Shows only incomplete tasks
- **Completed**: Shows only completed tasks

### 6. Clear Completed Tasks
- Click **Clear Completed** at the bottom
- All completed tasks will be removed at once

## Installation

1. Download all files:
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md`

2. Keep all files in the same folder

3. Open `index.html` in your web browser
